## 效果示例

> 以下是对话示例，展示各投资大师的思维方式如何分析实际问题

```
用户      ❯ 我有一笔钱想投资，现在该买股票还是持有现金？

巴菲特    ❯ 先别想买什么。先问自己三个问题：
            1. 这笔钱什么时候会用？
            2. 我能承受多大亏损？
            3. 我懂这个投资吗？

            如果三年内要用，就别进股市。
            如果亏损10%就睡不着觉，仓位太重了。
            如果你不懂这个企业，就别买这只股票。

            好机会是用不变应对万变。等待是一种投资动作。
```

```
用户      ❯ 怎么看现在的A股市场？

炒股养家   ❯ A股是政策市、情绪市、主力市。
            看盘不是看数字，是看资金的心。
            龙头股不倒，行情不了。龙头股倒了，小心。

            散户最大的问题是每天都想操作。
            真正的高手是80%的时间等待，20%的时间出手。
            一年抓住2-3次确定性机会就够了。
```

```
用户      ❯ 买入后股价暴跌50%，该怎么办？

利弗莫尔   ❯ 先问自己一个问题：买入时的逻辑还在吗？
            如果买入逻辑没变，跌是机会不是风险。
            如果买入逻辑变了，止损出场，不要犹豫。

            亏损出场后，不要立刻想翻本。
            等待下一次确定性机会。资本市场永远不缺机会，
            缺的是本金和耐心。
```

```
用户      ❯ 如何判断一只股票是被低估还是价值陷阱？

格雷厄姆   ❯ 三个标准：
            1. 市盈率低于行业平均的60%
            2. 市净率低于1.5
            3. 股价低于净流动资产的2/3

            低于这些标准不一定是机会，要看资产负债表是否健康。
            烟蒂股可能是价值陷阱，也可能是困境反转。
            关键看管理层是否靠谱，债务结构是否可控。
```

**这不只是在复读大师语录。** 巴菲特用的是「护城河+安全边际」框架，炒家用的是「情绪周期+龙头战法」，利弗莫尔用的是「趋势跟随+关键点突破」，格雷厄姆用的是「定量估值+定性验证」。**它们是用大师的认知框架帮你分析当下的问题。**

## 工作原理

本项目的制作方法论来自 [女娲.skill](https://github.com/alchaincyf/nuwa-skill)，核心是「认知操作系统」的提取与蒸馏。

**女娲的5层蒸馏法：**

| 层次 | 说明 |
|------|------|
| **怎么说话** | 表达DNA——语气、节奏、用词偏好 |
| **怎么想** | 心智模型、认知框架 |
| **怎么判断** | 决策启发式 |
| **什么不做** | 反模式、价值观底线 |
| **知道局限** | 诚实边界 |

**工作流程：**

1. **采集** — 并行采集大师的著作、访谈、演讲、决策记录
2. **提炼** — 提取心智模型（必须跨领域验证、有预测力、有排他性）
3. **构建** — 写入 SKILL.md：3-7个心智模型 + 决策启发式 + 表达DNA + 反模式 + 诚实边界
4. **验证** — 用大师公开回答过的问题测试，方向一致才通过

> 💡 **为什么要蒸馏投资大师？**
>
> 投资市场的钱，本质上是认知的变现。同样的K线图，10个人看到10种机会。
>
> 巴菲特看到的是企业价值，格雷厄姆看到的是价格与价值的差距，利弗莫尔看到的是趋势方向，索罗斯看到的是市场偏见。
>
> 学会用多种框架思考同一问题，是避免韭菜命运的第一步。

## 快速上手

### 新手入门路径

```
第1步 → 巴菲特/格雷厄姆    建立价值投资基础，理解"买股票就是买企业"
第2步 → 利弗莫尔           理解趋势和择时，建立交易思维
第3步 → 索罗斯             理解反身性和市场非理性
第4步 → 彼得林奇           学会从生活中发现投资机会
第5步 → 西蒙斯/德曼        理解量化思维和风险控制
第6步 → 炒股养家           理解A股特色和情绪周期
```

### 使用建议

- **问题导向**：遇到投资困惑时，调用对应大师的框架思考
- **对比学习**：同一问题，用不同大师的框架分析，对比差异
- **实战验证**：模拟盘用大师框架操作，记录复盘
