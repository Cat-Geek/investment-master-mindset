## 效果示例

> 以下是对话示例，展示8位投资大师的思维方式如何分析实际问题

---

### 1️⃣ 巴菲特

```
用户      ❯ 我有一笔钱想投资，现在该买股票还是持有现金？

巴菲特    ❯ 先别想买什么。先问自己三个问题：
            1. 这笔钱什么时候会用？
            2. 我能承受多大亏损？
            3. 我懂这个投资吗？

            如果三年内要用，就别进股市。
            如果亏损10%就睡不着觉，仓位太重了。
            如果你不懂这个企业，就别买这只股票。

            好机会是用不变应对万变。等待是一种投资动作。
```

---

### 2️⃣ 格雷厄姆

```
用户      ❯ 如何判断一只股票是被低估还是价值陷阱？

格雷厄姆   ❯ 三个定量标准：
            1. 市盈率低于行业平均的60%
            2. 市净率低于1.5
            3. 股价低于净流动资产的2/3

            但数字只是起点，不是终点。
            低于这些标准不一定是机会，要看资产负债表是否健康。
            烟蒂股可能是价值陷阱，也可能是困境反转。
            关键看管理层是否靠谱，债务结构是否可控。
            分散投资是格雷厄姆的底线——我不知道哪只是陷阱，所以我买一群。
```

---

### 3️⃣ 利弗莫尔

```
用户      ❯ 买入后股价暴跌50%，该怎么办？

利弗莫尔   ❯ 先问自己一个问题：买入时的逻辑还在吗？
            如果买入逻辑没变，跌是机会不是风险。
            如果买入逻辑变了，止损出场，不要犹豫。

            亏损出场后，不要立刻想翻本。
            等待下一次确定性机会。
            资本市场永远不缺机会，缺的是本金和耐心。

            趋势是你的朋友。顺势而为，不要逆势而动。
```

---

### 4️⃣ 索罗斯

```
用户      ❯ 为什么大部分散户在市场里赚不到钱？

索罗斯    ❯ 因为他们不理解反身性。
            市场参与者的看法会影响市场，市场又会反过来影响参与者。
            股价不只是反映基本面，它本身就是基本面的一部分。

            大部分人研究的是"市场是什么"，高手研究的是"市场正在变成什么"。
            这两个问题看起来差不多，实际上天差地别。
            前者是静态分析，后者是动态预判。
            当趋势形成时，不要问为什么，跟着走，直到拐点出现。
```

---

### 5️⃣ 彼得·林奇

```
用户      ❯ 普通上班族没时间研究股票，该怎么投资？

彼得·林奇  ❯ 恭喜你，你有一个巨大的优势——你每天都在消费。
            你知道哪家奶茶店排队最长，哪家店服务最好。
            这些信息比任何研报都有价值。

            买你熟悉的公司，买你生活里离不开的东西。
            不用盯盘，把钱放在那里，让公司为你工作。
            决定选股人命运的不是头脑而是耐力。
            拿住一个好公司，比来回折腾强一百倍。
```

---

### 6️⃣ 西蒙斯

```
用户      ❯ 短线交易能持续赚钱吗？

西蒙斯    ❯ 能。但不是你想的那种方式。
            我们不预测方向，我们寻找规律。
            市场有短期惯性，有可以被量化的价格模式。
            靠数学模型赚钱，不是靠 intuition。

            好处是：模型不情绪化，不恐惧，不贪婪。
            坏处是：模型会失效，需要不断迭代。
            成功的关键是——保持简单，不断测试，及时止损。
```

---

### 7️⃣ 德曼

```
用户      ❯ 量化模型真的能预测市场吗？

德曼      ❯ 不能。没有任何模型能预测市场。
            所有模型都是"好像"模型——它描述的不是真实世界，
            而是"一个可能存在的真实世界，是众多中的一个"。

            模型有两个致命敌人：
            一是模型的风险——你不知道模型什么时候会失效；
            二是人类的迷信——人们总以为模型能解释一切。

            我的建议是：相信模型，但保持怀疑。
            用模型赚钱，用智慧亏钱。
```

---

### 8️⃣ 炒股养家

```
用户      ❯ 怎么看现在的A股市场？

炒股养家   ❯ A股是政策市、情绪市、主力市。
            看盘不是看数字，是看资金的心。
            龙头股不倒，行情不了。龙头股倒了，小心。

            散户最大的问题是每天都想操作。
            真正的高手是80%的时间等待，20%的时间出手。
            一年抓住2-3次确定性机会就够了。

            做的是短线，但是看的是更大的局。
            先有大局观，才有短线操作。
```

## 工作原理

本项目的制作方法论来自 [女娲.skill](https://github.com/alchaincyf/nuwa-skill)，核心是「认知操作系统」的提取与蒸馏。

**女娲的5层蒸馏法：**

| 层次 | 说明 |
|------|------|
| **怎么说话** | 表达DNA——语气、节奏、用词偏好 |
| **怎么想** | 心智模型、认知框架 |
| **怎么判断** | 决策启发式 |
| **什么不做** | 反模式、价值观底线 |
| **知道局限** | 诚实边界 |

**工作流程：**

1. **采集** — 并行采集大师的著作、访谈、演讲、决策记录
2. **提炼** — 提取心智模型（必须跨领域验证、有预测力、有排他性）
3. **构建** — 写入 SKILL.md：3-7个心智模型 + 决策启发式 + 表达DNA + 反模式 + 诚实边界
4. **验证** — 用大师公开回答过的问题测试，方向一致才通过

> 💡 **为什么要蒸馏投资大师？**
>
> 投资市场的钱，本质上是认知的变现。同样的K线图，10个人看到10种机会。
>
> 巴菲特看到的是企业价值，格雷厄姆看到的是价格与价值的差距，利弗莫尔看到的是趋势方向，索罗斯看到的是市场偏见。
>
> 学会用多种框架思考同一问题，是避免韭菜命运的第一步。

## 快速上手

### 新手入门路径

```
第1步 → 巴菲特/格雷厄姆    建立价值投资基础，理解"买股票就是买企业"
第2步 → 利弗莫尔           理解趋势和择时，建立交易思维
第3步 → 索罗斯             理解反身性和市场非理性
第4步 → 彼得林奇           学会从生活中发现投资机会
第5步 → 西蒙斯/德曼        理解量化思维和风险控制
第6步 → 炒股养家           理解A股特色和情绪周期
```

### 使用建议

- **问题导向**：遇到投资困惑时，调用对应大师的框架思考
- **对比学习**：同一问题，用不同大师的框架分析，对比差异
- **实战验证**：模拟盘用大师框架操作，记录复盘
